Codes

Les bouts de code peuvent être insérés dans une ligne grâce à l’élément code mais également au sein d’un bloc de texte préformatté :

          /* ==================== *
             CHAT
           * ==================== */

          .liste-chat { list-style: none; }

          .item-chat {
            font-size: 13px; /* @affected IE8 */
            font-size: .8rem;
            padding: 1em 0;
          }

Un autre type de code peut être balisé :les raccourcis clavier. Par exemple, la plupart des navigateurs permettent de passer en plein écran en utilisant la touche F12. Vous le saviez ?
